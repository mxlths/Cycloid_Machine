# Cycloid Machine Documentation

This directory contains documentation for the Cycloid Machine project, which is designed to create complex geometric patterns (cycloids) through the coordinated movement of multiple rotating wheels.

## Contents

1. [Project Overview](./project_overview.md) - Overview of the entire project
2. [Hardware Components](./hardware_components.md) - Details about the physical components of the machine
3. [Control System](./control_system.md) - Information about the Arduino control system
4. [Menu System](./menu_system.md) - Details about the LCD menu interface
5. [Python Simulator](./python_simulator.md) - Documentation for the LCD Menu simulation

## Current Project Status

The project is currently in development with the following components defined:
- Bill of Materials (BOM) for all hardware components
- Design specifications for the LCD menu system
- Arduino pin definitions
- Python simulator for the LCD menu interface

The core Arduino code for controlling the motors appears to be pending implementation.
